First-year Electronics and Communication Engineering student at MSIT, GGSIPU.

I like building things that combine hardware and software. Most of my work revolves around embedded systems, IoT, robotics, and electronics. I'm particularly interested in understanding how systems work from the sensor level all the way to the software layer.

---

## Projects

### ARIA Agri-Rover

An autonomous environmental monitoring rover built around the ESP32.

The project integrates multiple sensors using I2C, ADC, and 1-Wire communication, collects field data, and sends telemetry through MQTT to a Flask backend. Data is stored in SQLite and displayed through a live dashboard.

**Tech:** ESP32, C++, Python, MQTT, Flask, SQLite

---

### RFID Attendance System

An RFID-based attendance system built using ESP32 and RC522 modules.

The project handles card scanning, UID decoding, attendance tracking, and runs on custom firmware written in C++. A custom enclosure was also designed and fabricated for demonstrations.

**Tech:** ESP32, RC522, C++, SPI

---

### UniScout

A college discovery and comparison platform.

The goal was to make information about colleges easier to compare through searchable listings, filters, and placement-related data.

**Tech:** JavaScript, Tailwind CSS, Git, Vercel

---

### Guitar Distortion Pedal

A multi-stage analog audio effects circuit designed and assembled from scratch.

The design includes an input buffer, filter stage, soft-clipping distortion stage, and output buffer. The circuit was tested and validated using an oscilloscope.

**Tech:** Analog Electronics, Op-Amps, Circuit Design
